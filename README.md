# Apache Lucene (apache-lucene)
Apache Lucene is a high-performance, full-featured text search engine library written entirely in Java. It provides indexing and search technology, as well as spellchecking, hit highlighting, and advanced analysis/tokenization capabilities.

### Apache Lucene
Lucene provides a comprehensive Java API for full-text indexing, searching, faceting, hit highlighting, spatial search, and text analysis with support for custom analyzers, query parsers, and scoring.
